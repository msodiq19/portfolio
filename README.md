# dmsodiq.xyz — Personal Portfolio

Personal portfolio website for **Muhammed Sodiq**, a software engineer focused on systems, architecture, and security-minded development.

**Live:** [dmsodiq.xyz](https://dmsodiq.xyz)

## Tech Stack

- **Framework:** Next.js 16 (App Router, React Compiler)
- **Language:** TypeScript
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
